# 技能包示例（skill-package-demo）

本目录提供符合上传校验的 **`skill.yaml` + `README.md`**，并附带 **`CHANGELOG.md`**（可选，一并打入 `demo.zip` 以演示完整交付形态）。

## 内容说明

- **`skill.yaml`**：字段齐全、含注释分隔线；`slug` 默认为 `skill-hub-package-demo`，上传前请改为未占用 slug。
- **`README.md`**：扩展版说明——概述、根目录文件表、`skill.yaml` 字段表、投稿自检清单、FAQ、版本引用。
- **`CHANGELOG.md`**：示例变更记录，可与版本号同步维护。

## 使用方式

1. 复制本目录内文件，按实际技能修改（尤其 `slug`、`version`、`category`）。
2. 打包示例（含可选 `CHANGELOG.md`）：
   ```bash
   cd examples/skill-package-demo
   zip -r demo.zip skill.yaml README.md CHANGELOG.md
   ```
3. 也可直接使用同目录下的 **`demo.zip`** 联调上传；若提示 slug 冲突，请修改 `skill.yaml` 后再传。

## 前台 / 后台页面下载

开发或部署后，可在**技能广场上传中心**或**后台上传技能包**弹窗内点击「下载示例 ZIP」，静态地址为 **`/downloads/skill-package-demo.zip`**（由 `apps/portal-web/public` 与 `apps/admin-web/public` 提供）。更新示例时请同步替换这两处文件与仓库 `examples/skill-package-demo/demo.zip`。

## 字段说明

以服务端校验为准：`name`、`slug`、`version`（semver）、`summary`、`description`、`category`（分类 slug）、`tags`（数组）、`author`。
